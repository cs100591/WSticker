# Product Overview

**Clasp** (formerly DailyPA) is a cross-platform mobile productivity app for iOS and Android built with React Native and Expo.

## Core Features

- **Todo Management**: Create, track, and organize tasks with priorities, colors, and due dates
- **Expense Tracking**: Log expenses with categories and OCR-powered receipt scanning
- **Calendar Integration**: Manage events with automatic todo-to-event conversion and native iOS calendar sync
- **AI Assistant**: Natural language interface for creating todos and expenses via chat
- **Offline-First**: Full offline support with automatic bidirectional sync when online

## Key Capabilities

- Biometric authentication (Face ID/Touch ID)
- Sign in with Apple (iOS)
- Push notifications with deep linking
- Receipt OCR extraction
- Real-time statistics and analytics
- Theme support (light/dark/system)
- Multi-currency expense tracking
- Auto-lock with configurable timeout

## Architecture Philosophy

- **Offline-first**: Local-first data storage with background sync
- **Security-focused**: Encrypted storage, HTTPS-only, secure logging
- **Cross-platform**: Single codebase for iOS and Android
- **Type-safe**: TypeScript throughout with strict mode
- **Testable**: Property-based testing with fast-check

## User Experience

The app prioritizes seamless offline functionality, automatic data sync, and secure authentication. Users can work without connectivity and changes sync automatically when online.
